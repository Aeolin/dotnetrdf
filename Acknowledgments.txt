      <p class="repo-desc-description">dotNetRDF is a API for RDF written in C# 3 under .Net Framework 3.5 It allows for reading/writing of RDF in all common formats, querying over RDF data in various ways and supports a variety of different mainstream Triple stores. It also includes a SPARQL 1.1 query and update engine. For persistent storage it supports AllegroGraph, 4store, Joseki, Fuseki, Sesame, Stardog Talis and Virtuoso</p>
